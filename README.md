## Blog_App

This is a Ruby on Rails website that will show the list of posts,give comments and like the post

## Built with

Ruby
Rubocop
Stylelint
RoR

## SetUp
To get a local copy up and running follow this steps

git clone https://github.com/IfzaRasool/blog_app.git
Run bundle install
Run the project
rails s
